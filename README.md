# CS4379 Assignment 1 - Netflix Titles Analysis

## Description
This repository contains the notebook `analysis.ipynb` for Assignment 1 of CS4379.  
In this notebook, we analyze the **Netflix titles dataset** to understand how the number of titles added to Netflix has changed over time and to explore the distribution of release years in the catalog.
